Finanzas App

Aplicación web de gestión financiera personal que permite llevar control de ingresos, egresos, presupuestos y visualizar estadísticas en tiempo real con gráficos interactivos.

🚀 Tecnologías
HTMLCSS PHPMySQL
✨ Características
  • Dashboard con estadísticas en tiempo real
  • Categorización automática de gastos
  • Administrar movimientos
  • Consultas generales en base al movimiento
  • Interfaz de usuario intuitiva y registro unicos por usuarios
  • Historial de acciones
  • Informacion del usuario logueado
  • Registro de usuario a través de formulario y confirmacion de gmail
🎯 Problema resuelto

Muchas personas pierden el control de sus finanzas por falta de herramientas simples e intuitivas. Esta app centraliza toda la gestión financiera en un solo lugar visual y fácil de usar.

+QueCafe

Plataforma web para cafeterias artesanales que combina menu digital interactivo, facil adquisicion de productos y manejo de los mismos, y no mejor importante fidelizacion con nuestros clientes.

Tecnologias
HTMLCSS PHPMySQL
Caracteristicas
  • Menu digital con imagenes HD
  • Catalogo de productos
  • Sistema de pedidos online
  • Programa de puntos y recompensas
  • Dashboard 100% funcional para cualquier tipo de necesidad
  • Vista y interacciones rapidas y sencillas
Problema resuelto

Las cafeterias artesanales carecian de presencia digital moderna. +QueCafe les dio una solucion todo-en-uno que mejoro la experiencia del cliente y redujo tiempos de espera y adquisicion de productos muchos mas comodos para el cliente.
